如何不使用第三方网站帮助国内用户创建Steem帐户

作者：子衿 2017-08-16
  • Steem
  • Steemit
  • 如何帮助国内用户创建Steem帐户


之前 @lemooljiang 和 @jubi 分别写过一篇文章，谈到可以使用第三方网站／工具，比如steemconnect来创建新的账户，不过需要把你的active key提供给steemconnect。这个让羽西感觉很不爽，因为用习惯了OAuth之类的系统，很不习惯把自己的密码或者key提供给第三方。因此，察看了一下官方文档，其实完全可以自己帮别人创建帐户。整个过程也是非常快的，只需要几分钟的时间。

获取一个VM

首先，你需要有一个自己的VM (Virtual Machine)，这个可以是本地的VM，也可以是一些主机商的VPS。比如：羽西就有自己开发专用的Vultr VPS。如果你没有VM，甚至不知道什么是VM，也不要紧。根据前面总结过的步骤小白也可以做到。

简单总结一下：

1) 创建一个Codenvy账户：https://codenvy.io/
2) 创建一个Python workspace
3) 运行以下命令安装Steem Python库 

sudo pip3 install steem

帮助国内朋友创建Steem账号

如果想要帮助国内朋友创建Steem账户，需要花费6个STEEM，所以首先确保你的账户STEEM足够才行。这6个STEEM在账号创建成功后会自动转到你朋友的账户里。

具体的创建步骤如下：

首先，在前面创建好的VPS中键入如下命令导入你的steem账户：

steempy importaccount 你的账户名

上图中的第一个”Account Passphrase”指的是你的Steemit登陆用的密码。而第二个”Passphrase”则是后面创建新账户时需要输入的密码。这个密码必须输入两次确认才行。

运行以下命令指定默认账户：

steempy set default_account 你现在帐户

接下里就可以创建新的账户了！输入如下命令：

steempy newaccount 新账户名

系统会要求你输入三次密码。前两次为新账户的密码。最后一次则是前面导入账户时的密码。新创建账户的密码一定要使用一个复杂不易记忆的密码。

至此大功告成！
