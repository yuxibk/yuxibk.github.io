Ledger Nano S冷钱包设置及使用

作者：子衿 2018-11-27
  • Ledger
  • Ledger Nano S
  • 冷钱包


接触区块链和加密货币时间长了，就要开始考虑自己的资金安全性问题了。那么，硬件钱包自然是不二的选择。下面结合自己的经验给大家详细介绍一下如何设置Ledger Nano S硬件钱包，如何使用Ledger Nano S存储比特币等加密货币，以及相关的注意事项。

硬件钱包为什么更安全？

硬件钱包也叫做冷钱包，其实说白了就是把你的数字货币资产离线存储，实现和网络的物理隔绝，这样再厉害的黑客也无计可施了。在区块链上进行交易的时候，我们需要用自己的私钥来签署交易。那么，如果我们的私钥存放于电脑或网络中，就有可能会被黑客获取，进而导致资产的丢失。而硬件钱包的原理是借助于24个单词生成私钥，且私钥只存储于冷钱包中。需要交易后，只是由冷钱包将签署后的交易发送到电脑，这样就最大限度的保证了私钥的安全。而通常的建议是将那24个单词写在纸上，和冷钱包物理隔离。

Ledger Nano S钱包介绍

Ledger就是一家法国专门生产冷钱包的公司，他家旗下的产品有Ledger Nano S 和 Ledger Blue。这里重点给大家推荐我自己也在用的Ledger Nano S。截至到我写这篇文章，Ledger Nano S目前支持包括比特币，以太坊，比特币现金，瑞波币等712种加密货币。

如何设置Ledger Nano S 钱包

首先从官网下载Ledger Live: https://www.ledger.com/pages/ledger-live

安装Ledger Live，之后运行该程序，主界面为：

选择他家适合的产品，我这里选择“Ledger Nano S”：

看到下面这个界面后将Ledger Nano S连接到电脑：

连接后同时按下两个按钮，之后设置一个8位的PIN。两个键同时按下代表确认，左/右键分别向下/上翻，确认PIN后Ledger Nano S会提示要保存好PIN。

进入下一步，再纸上手工记录24个单词并再次确认。Ledger Nano S会提示”Your device is now ready.“

这里建议大家先尝试这用这24个单词恢复一下钱包，确认没有问题后再把加密货币转进来。

对于记录这24个单词的纸，一定要保管好！一定要保管好！一定要保管好！一旦Ledger Nano S丢失或损坏，就要靠他们来恢复你的账号了！一旦丢失，你的资产就再也找不回来了。
