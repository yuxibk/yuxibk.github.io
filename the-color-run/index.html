英国The Color Run彩色跑介绍

作者：至今 2017-03-01
  • The Color Run
  • 伦敦
  • 布莱顿
  • 彩色跑
  • 快乐五公里
  • 曼彻斯特


今天羽西和大家来说说这个”The Color Run”，中文叫”彩色跑”。The Color Run现在是一个火遍全球的公共活动，也叫做“快乐五公里”。Color Run在中国都有了，英国人当然玩得更High。。。其实，它和Run没什么关系。整个活动中对参与者的速度没有任何要求，当然，这也不是比赛。其实你可以把The Color Run理解成一个大的主题Party。你可以跑，可以跳，可以走，甚至可以跳舞。The Color Run最为强调的是要快乐，体现自我。

The Color Run更注重在整个活动中制造的乐趣。活动全长5公里，每到一公里处都会有尽职的工作人员往参赛者身上喷洒各种涂料，保证每个人都会色彩斑斓，绝对可以和孔雀去PK。当然，到了色彩站你也不能客气喔，拿起涂料，对准其他参与者，发挥你创意的时刻到了。。。不用担心，官方提供的这些彩粉都是由100%纯天然玉米淀粉和天然色素制成，绝对健康环保，保证健康好看但不保证好吃~~~~如果你担心粉末进入眼睛或者呼吸道，也可以准备好墨镜和口罩，不过这样乐趣就少了很多喔。。。

在大家到达终点后，终点的舞台上还会有一个色彩的Party，大家会把手中的涂料粉洒向空中，非常壮观！

2017年的The Color Run会在以下三个英国城市举行，具体细节如下：

伦敦：2017年6月11日
曼彻斯特：2017年7月1日
布莱顿：2017年9月23日

The Color Run肯定是一次欢乐的体验，赶快加入地球上最欢乐的5公里”吧！现在The Color Run优惠票开始预售啦，还可以获赠T-shirt和墨镜。
