使用ImprovMX和gmail配置自己域名的免费邮件服务

作者：子衿 2019-11-08
  • 邮件服务器


在拿到一个新域名时，配置相应的邮件服务器是必不可少的一个步骤。由很多免费的邮件系统可以使用，比如：Zoho Mail, Migadu, Yandex, 和 ImprovMX. 下面介绍一下如何使用ImprovMX 和 Gmail配置自己的个性化域名邮件.

ImprovMX

打开 ImprovMX 邮件转向服务。添加如下的 MX 记录:

Host    Value                  Priority
----------------------------------------------
@       mx1.improvmx.com       10
@       mx2.improvmx.com       20

添加想要配置的域名以及你的个人gmail，然后进行验证。

配置gmail

进入如下页面创建APP密码：https://security.google.com/settings/security/apppasswords

进入gmail配置页面，然后：Accounts and Import -> Send mail as -> Add another email address

取消 “Treat as alias” 选项

使用 “smtp.gmail.com” 作为 SMTP server, 在 Username项中填入自己的gmail, 在 password项中填入前面生成的APP password:

单击 “Add Account”

最后登录自己邮箱获取验证码完成验证。
