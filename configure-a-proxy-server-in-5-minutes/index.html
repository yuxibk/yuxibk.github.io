5分钟配置一个Ubuntu代理服务器

作者：子衿 2016-10-31
  • 代理服务器


我想代理服务器的作用大家都懂，我就不多说了，这里给大家介绍一下如何快速的配置一个代理服务器。当然前期是你需要有一个国外的VPS。

申请一个国外VPS服务器

关于好用的VPS，羽西专门写过一篇文章，大家可以看看：推荐几个好用的国外VPS

个人强烈推荐Vultr家的VPS,　最便宜的才需要 $5一个月。而且点我的链接注册还能送$20，相当免费使用4个月。

安装代理服务器软件

apt install python-gevent python
apt-get install python-pip
pip install shadowsocks

编辑配置文件:

nano /root/proxy/config.json

配置文件内容如下:

{
  "server":"你的VPS服务器IP",
  "server_port":8888,
  "local_port":1080,
  "password":"你的密码",
  "timeout":600,
  "method":"aes-256-cfb"
}

运行服务器软件:

nohup ssserver -c /root/proxy/config.json

安装配置客户端软件

在以下网址下载客户端软件，以Windows版为例，只需要下载zip文件并解压缩就可以直接运行了: https://shadowsocks.org/en/download/clients.html

按照下图所示的配置一下客户端:

右键单击程序图标然后选中 ‘Enable system Proxy’

好了，设置完毕，你可以浏览Google, facebook等网站了。。。
