Steemit中的密码和各种key

作者：子衿 2017-09-18
  • Steem
  • Steemit


很多新手在刚接触Steemit的时候可能对于Steemit的密码和各种key比较困惑，下面把自己的理解和大家分享一下。

Steemit的密码

如果你是在Steemit注册获得的账户，系统会自动给你创建一个复杂的密码，如果是别的朋友帮你创建的账户，这个密码是你或者你的朋友创建的。这个密码是可以改的，登陆Steemit后，单击右上角的那三个横道的图标后，从菜单中选择”Change Account Password”就可以更改你账户的密码。

在改密码的时候，一定要选择一个复杂的密码，Steemit本身也能自动生成密码。同时你可以选择第三方网站来生成密码。这里推荐两个网站，建议把两个网站自动生成的密码组合并更改一下，增大随机性，这样就更靠谱了。

https://passwordsgenerator.net/
https://lastpass.com/generatepassword.php

至于生成了复杂密码后如何保存，请参照我前面写过的文章

另外，Steemit的密码除了用来登陆Steemit外， 其他操作，比如，发帖，赞，转账等等等，用下面要说的各种key就都可以搞定。因此，你不需要用Steemit密码登陆任何第三方网站。

关于Steemit的各种key

单击 “Wallet” -> “Permissions”，你就会看到各种key：

关于各种key的作用，很多人都已经总结过了，我这里就简单总结一下：

  • POSTING key: 用来发帖和赞的。
  • ACTIVE key: 用来进行资金操作的，比如转入和转出你的SBD或者STEEM，还有买入和卖出你的资金时。
  • OWNER key: 这个key具有所有权限，包括修改OWNER key自己。
  • MEMO key: 官方说法是用来创建和管理MEMO的，这个没有用过。

你可以看到，每个key后面都有一串长长的数字，对应的就是public key，也就是公钥，这个公钥在整个区块链网络上对所有人都可见，自然也不能用来进行任何操作的。它的作用主要用来完成数字签名的认证，确保所有交易和操作的安全性。

要想进行任何操作，都需要获取相应的私钥，也就是private key才行。对于POSTING key，单击“SHOW PRIVATE KEY”就可以看到。而ACTIVE KEY的私钥则需要单击”LOGIN TO SHOW”并登陆后才能看到。

这里可以用你的密码或者OWNER key登陆都可以。

关于靠谱的第三方网站

有些Steemit的APP/网站要求输入你的账户信息，这个时候建议只使用那些key，不要输入你的密码。而且尽量提供权限最小的key，比如：如果Steemit APP只是发帖/点赞的话就只提供POSTING key，而不要提供ACTIVE key。

另外，怎样甄别一个APP是否靠谱呢？

最直接的办法就是看看这个APP是否受到Steemit官方认可。由于Steemit是开源软件，你可以访问它们github上的源代码，看看你打算使用的网站/APP是否在官网列表中，比如：很多人都在用的Steemconnect就在这个列表中，因此可以放心的使用。
